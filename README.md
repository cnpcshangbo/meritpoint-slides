# MeritPoint slides — static backup mirror

**Live mirror:** https://cnpcshangbo.github.io/meritpoint-slides/

Auto-generated backup of the deck site at courses.hopeembark.org, for use
when the primary server is unreachable during class. Do not edit by hand —
regenerate with `courses-site/deploy-mirror.sh` in the (private) meritpoint repo.

Static-only: slides, quizzes, in-browser Python labs, narration audio, and
PDFs all work; per-slide comments and the AI lab assistant need the primary
server and degrade gracefully here.
